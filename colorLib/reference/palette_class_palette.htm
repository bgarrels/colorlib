Name
Palette
Examples
None available
Description
Creates a Palette object which will hold all your colors.
Syntax 
Palette(i_p);
Palette(i_p, i_length);
Palette(i_p, i_colors);
Palette(i_p, i_image);
Palette(i_p, i_image, cnt);
Palette(i_p, i_colors, cnt);
Palette(i_p, i_filename);
Parameters
i_p
PApplet, normally use 'this'
i_length
int: length of the created
i_colors
color[]: an array of colors
i_image
PImage:
cnt
int: the count of colors that will be extracted from the pic using the medain cut algorithm
i_filename
String: an .cs or .act file located in the data folder of your sketch
Methods
addColor ( )
Add one color at the end of your palette.
addColors ( )
Add all passed colors at the end of your palette.
darken ( )
Decrease the brighness of all colors in the palette by the passed value. By default its 10.
deleteDuplicate ( )
Delete all duplicate colors within the palette.
desaturate ( )
Increase the saturation of all colors in the palette by the passed value. By default its 10.
drawSwatches ( )
Draw all colors
drawWheel ( )
Draws a color wheel with all colors of the palette.
getAverage ( )
Returns the average color of the palette.
getColor ( )
Returns the color at the given position in the color array of the palette.
getColors ( )
Returns an array holding all the color of the palette.
getDarkest ( )
Returns the darkest color of the palette.
getLightest ( )
Returns the lightest color of the palette.
getNearestHues ( )
Returns an array with the nearest hue to the palettes color as an array of the following strings: "red", "orange", "yellow", "lime", "green","teal", "cyan", "azure", "blue", "indigo", "purple", "pink"
getSwatch ( )
Returns the swatch at the given position in the color array of the palette.
getSwatches ( )
Returns an array holding all the swatches of the palette.
interpolate ( )
Interpolates all colors of the palette with passed color.
lighten ( )
Increase the brighness of all colors in the palette by the passed value. By default its 10.
makeComplement ( )
Creates a 2 color palette with the passed color and it's complement on the RYB color wheel.
makeComplementary ( )
Creates a 6 color palette with the passed color and it's complement on the RYB color wheel and 2 darker and softer contrasting colors of each complement.
makeSplittedComplementary ( )
Creates a 3 color palette with the passed color and it's left and right complement on the RYB color wheel.
makeTetrad ( )
Creates a 4 color palette that make up an cross on the RYB color wheel.
makeTriad ( )
Creates a 3 color palette that make up an triangle on the RYB color wheel. The default angle of the triangle is 120 degrees, for an equilateral triangle, but you can pass your own in the range of 0-180.
rotateRGB ( )
Rotates all colors by the passed angle on the RGB color wheel.
rotateRYB ( )
Rotates all colors by the passed angle on the RYB color wheel.
saturate ( )
Increase the saturation of all colors in the palette by the passed value. By default its 10.
save ( )
Save the palette as an or .act file in your sketch folder
setColor ( )
Sets the color on the passed position on palette.
setColors ( )
Delete all colors of the palette and fill the palette with the passed colors.
sortByHue ( )
Sorts the palette by hue starting with red end ends blue.
sortByLuminance ( )
Sorts the palette by luminance with element at last index containing the "brightest" item of the palette.This method based on toxis color code snippets.
sortByProximity ( )
Sorts the palette by proximity to a colour with element at first index containing the "closest" item of the palette.This method based on toxis color code snippets.
sortBySaturation ( )
Sorts the palette by saturation with element at last index containing the most saturated item of the palette. This method based on toxis color code snippets.
totalSwatches ( )
Returns the count of all swatches in the Palette
