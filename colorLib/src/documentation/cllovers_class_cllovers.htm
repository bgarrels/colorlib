Description

Syntax

ClLovers(i_p);

Parameters

i_p
Methods
getBriRange ( )
Returns the brightness range.
getColors ( )
Returns the colors for a given array of keywords.
getHueRange ( )
Returns the hue range.
getNumResults ( )
Returns the number of results.
getPalettes ( )
Returns the palettes for a given array of keywords.
getResultOffset ( )
Returns the result offset.
printXML ( )
Use this method to print the resulting XML in the console.
setBriRange ( )
Sets the brightness range to query the COLOURLovers API. This function takes a int[] array with a size of 2. The values of the integers should be in the range of 0 to 99.
setHueRange ( )
Sets the hue range to query the COLOURLovers API. This function takes a int[] array with a size of 2. The values of the integers should be in the range of 0 to 359.
setNumResults ( )
Sets the number of results to query. The value of the integer should not be bigger than 100.
setResultOffset ( )
Sets the result offset used to query the COLOURLovers API.
setSortBy ( )
This function sets how the colors should be sorted. Allowed values are ASC for ascending and DESC for descending.
