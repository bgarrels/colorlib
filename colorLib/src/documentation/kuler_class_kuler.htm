Name

Kuler

Examples

None available

Description

The Kuler object is a container to query the Adobe Kuler service and get the response as a KulerTheme object. Take a look at the Kuler API to see the possibilities of the Kuler service.
A Kuler palette contains 5 colors per theme. The default number of themes returned per query is 20, but you can increase the count up to 100. To query more than 100 themes, you have to increase the startIndex. A query with maxItems=100 and startIndex=2 will return 100 themes starting at 200.

Syntax

Kuler(parent);

Parameters

parent
PApplet: normally you will be using new Kuler(this);"
Methods
getHighestRated ( )
Get the highest rated colors as an array of KulerThemes.
getMaxItems ( )
Get the maximum number of items to display on a page, in the range [1..100]. Default is 20.
getPopular ( )
Get the most popular colors for the specified number of days (default is 30 days) as an array of KulerThemes.
getRandom ( )
Get random colors as an array of KulerThemes.
getRecent ( )
Get the most recent colors as an array of KulerThemes.
getStartIndex ( )
A 0-based index into the list that specifies the first item to display. Default is 0, which displays the first item in the list.
printXML ( )
Use this methode to print the result xml in the console.
search ( )
Use your own query string or creates one by a given filter name and a query. Possible filters are "themeID", "userID", "email", "tag", "hex" and "title". Take a look at the API to see the possibilities of the kuler service.
The result is an array of KulerThemes.
setKey ( )
Sets the Developer Key needed to talk to the Kuler API.
setMaxItems ( )
Set the maximum number of items to display on a page, in the range [1..100]. Default is 20.
setStartIndex ( )
A 0-based index into the list that specifies the first item to display. Default is 0, which displays the first item in the list.
setserverPage ( )
As you cant connect to the kuler service directly if you run your sketch as an applet, you can set a path to a php or whatever page that's querry the kuler service and response the xml to the applet.
