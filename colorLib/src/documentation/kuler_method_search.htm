Name

search ( )

Examples

import colorLib.*;
import colorLib.webServices.*;

void setup(){
  size(400, 400);
  background(255);
   Kuler kuler = new Kuler(this);
  KulerTheme[] kulerThemes = kuler.search("tag", "germany");
  for(int i=0; i<kulerThemes.length; i++){
    pushMatrix();
    translate(10+(i%3)*130 , 10+floor(i/3)* 70);
    kulerThemes[i].drawSwatches();
    popMatrix();
  }
}

Description

Use your own query string or creates one by a given filter name and a query. Possible filters are "themeID", "userID", "email", "tag", "hex" and "title". Take a look at the API to see the possibilities of the kuler service.
The result is an array of KulerThemes.

Syntax

search(searchQuery);
search(query, filter);

Parameters

searchQuery
String: your query
query
String: query that is use with a searchFilter
filter
String: one of the following filters: "themeID", "userID", "email", "tag", "hex" and "title"

Returns

KulerTheme[], an array of all themes which match the query
