Name
Kuler
Examples
None available
Description
The Kuler object is an container to query adobes kuler service and get the response as an KulerTheme object. Take a look at the API to see the possibilities of the kuler service.
At kuler the palette a maximal count of 5 colors per theme. The default size of the themes per query is 20, but you can increase the count up 100. To query more then 100 you have to increase the startIndex. So a query with maxItems=100 and startIndex=1 will give you themes fromm count 100 to 200.
Syntax
Kuler(parent);
Parameters
parent
PApplet: typically use "this"
Methods
getHighestRated ( )
Get the highest rated colors. Use getThemes() to get the result as an array of kulerThemes.
getMaxItems ( )
Get the maximum number of items to display on a page, in the range [1..100]. Default is 20.
getPopular ( )
Get the popular colors for the specified number of days (default is 30 days). Use getThemes() to get the result as an array of kulerThemes.
getRandom ( )
Gets random colors. Use getThemes() to get the result as an array of kulerThemes.
getRecent ( )
Get the most recent colors. Use getThemes() to get the result as an array of kulerThemes.
getStartIndex ( )
A 0-based index into the list that specifies the first item to display. Default is 0, which displays the first item in the list.
getThemes ( )
Returns all themes of the Kuler object. Every query result is stored in the object. So calling first getRecent() and then getRandom() will give you an array with 40 max items.
search ( )
Use your own query string or creates one by a given filter name and a query. Possible filters are "themeID", "userID", "email", "tag", "hex" and "title". Take a look at the API to see the possibilities of the kuler service.
Use makePalettes() to get the result as an array of kulerThemes.
setMaxItems ( )
Set the maximum number of items to display on a page, in the range [1..100]. Default is 20.
setStartIndex ( )
A 0-based index into the list that specifies the first item to display. Default is 0, which displays the first item in the list.
