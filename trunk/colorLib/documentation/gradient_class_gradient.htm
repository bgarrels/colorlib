Name
 
Gradient
 
Examples
 
  
None available
 
 
Description
 
Create a palette with all colors of a gradient. The class extends the Palette class, so all methods of the Palette class can accessed from here.
 
Syntax
 
  
Gradient(i_p, i_colors, i_size);
Gradient(i_palette, i_stepSize);
Gradient(i_p, i_colors, i_size, i_wrap);
Gradient(i_palette, i_size, i_wrap);
 
 
Parameters
i_p
i_colors
int[], an int array which holds the colors in the processing way
stepSize
int, the size of the gradient
i_palette
Palette, creates the gradient from the colors of palette swatches
i_wrap
boolean, if its true the gradient ends with the same colors as its begin (default is false)
i_palette
i_size
i_wrap
