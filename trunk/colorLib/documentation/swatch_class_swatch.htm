colorLib.webServices

index
Name
Swatch
Examples
None available
Description
Creates a swatch object with the overloaded or a random color. This swatch object holds all information on a color. All color transformations are done on this
Syntax
Swatch(i_p);
Swatch(i_p, color);
Parameters
i_p
PApplet, normally you will use 'this'.
color
color, color
Methods
black ( )
Gets the black value for the color of this swatch. Based on the generic RGB to CMYK formula: black = minimum(1 - red, 1 - green, 1 - blue)
brightnessDiff ( )
Calculated the brightness difference between the swatch and the given color or swatch. The brightness is calculated by the following formular suggest by the W3C: ((Red value * 299) + (Green value * 587) + (Blue value * 114)) / 1000 The result should be higher then 125 for a good readability.
colorDiff ( )
Calculated the color difference between the swatch and the given color or swatch by the following formular suggest by the W3C: (maximum (Red value 1, Red value 2) - minimum (Red value 1, Red value 2)) + (maximum (Green value 1, Green value 2) - minimum (Green value 1, Green value 2)) + (maximum (Blue value 1, Blue value 2) - minimum (Blue value 1, Blue value 2)) The result should be higher then 500 for a good readability.
cyan ( )
Gets the cyan value for a given RGB color. Based on the generic RGB to CMYK formula: cyan = (1 - red - black) / (1 - black)
darken ( )
Darkens the color by a given number. The deafult is 10.
desaturate ( )
Desaturates the color by a given number. The deafult is 10.
getColor ( )
Returns the color of the swatch.
getNearestHue ( )
Returns the nearest hue to the color as an one of the following strings: "red", "orange", "yellow", "lime", "green","teal", "cyan", "azure", "blue", "indigo", "purple", "pink"
lighten ( )
Lightens the color by a given number. The deafult is 10.
magenta ( )
Gets the magenta value for a given RGB color. Based on the generic RGB to CMYK formula: magenta = (1 - green - black) / (1 - black)
rotateRGB ( )
Rotates a color around the RGB (Red-Green-Blue) color wheel.
rotateRYB ( )
Rotates a color around the RYB (Red-Yellow-Blue) color wheel. More information about the RYB color wheel can be found at Wikipedia.
saturate ( )
Saturates the color by a given number. The deafult is 10.
setColor ( )
Sets the color of the swatch
yellow ( )
Gets the yellow value for a given RGB color. Based on the generic RGB to CMYK formula: yellow = (1 - blue - black) / (1 - black)
